# 🚀 AI Content Monetization Agent

一个基于多Agent协作的AI自动内容生成与变现系统，支持热点分析、内容生成与爆款优化，适用于小红书、公众号等新媒体平台。

---

## 🔥 项目亮点

- 🧠 多Agent协作（趋势分析 / 内容生成 / 爆款优化）
- ⚡ 自动化内容生产（每日50+篇）
- 📈 提升曝光率与点击率
- 🔁 完整闭环：分析 → 生成 → 优化 → 输出

---

## 🧩 系统架构
用户输入主题
↓
趋势分析 Agent
↓
内容生成 Agent
↓
爆款优化 Agent
↓
内容输出（批量生成）

## 🚀 快速开始

```bash
pip install -r requirements.txt
python main.py
🧠 核心逻辑

系统采用多Agent协作机制：

Trend Agent：热点分析 + 用户痛点识别
Content Agent：基于长链推理生成内容
Rewrite Agent：爆款优化，提高传播效果

形成完整自动化闭环。
